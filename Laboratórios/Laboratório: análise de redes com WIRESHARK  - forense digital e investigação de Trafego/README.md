# Pentest Lab: Análise de Redes com Wireshark

## Resumo
- **Data:** Junho/2026  
- **Duração:** ~5 horas  
- **Objetivo:** Analisar tráfego de rede simulado (`.pcap`) para identificar servidores, credenciais, exfiltração de dados e vulnerabilidades.  
- **Habilidades:** Filtros Wireshark, análise de protocolos (DNS, HTTP, SMTP, Telnet, SMB), reconstrução de fluxos, exfiltração via DNS, decodificação de dados, identificação de falhas de segurança.

## Contexto
Simulação de uma empresa fictícia (Pacific Corp) com tráfego misto, incluindo tentativas de invasão, credenciais em texto puro e exfiltração de dados via DNS.

## Estrutura do Lab
- `/pcap` — arquivo de captura (anonimizado)  
- `/screenshots` — evidências (redigidas)  
- `/filtros` — lista de filtros utilizados  
- `/decodificacao` — scripts e ferramentas  
- `report.pdf` — relatório final  
- `findings.md` — vulnerabilidades encontradas
